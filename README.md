Based on tutorial from: [https://www.sitepoint.com/build-chatgpt-clone-react-openai-api/](https://www.sitepoint.com/build-chatgpt-clone-react-openai-api/)

Other features added:

- history
- formatting
- ollama intrgration
- ui changes
